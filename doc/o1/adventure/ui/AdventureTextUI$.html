AdventureTextUI

o1.adventure.ui.AdventureTextUI
object AdventureTextUI extends App

The singleton object AdventureTextUI represents a fully text-based version of the Adventure game application. The object serves as an entry point for the game, and it can be run to start up a user interface that operates in the text console.
